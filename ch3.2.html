3.2 Types of Access

This section will briefly explian two main methods to access data: Serial Access & Direct Access

3.2.1 Serial Access

Serial Access are files which are stored in a sequence, it will start from the beginning and search through each file one by one to find the file needed. It is usually used in situations where speed is not necessary such as bank cheques.

Ex. Magnetic Tape, VHS, Cassette tape

3.2.2 Direct Access

Direct access are files which do not have to be sorted and searched through to find the file needed like the serial access. These files are accessed directly, just like the name, and they are usually used in real-time applications where speed is vital.

Ex. CDs, DVDs, HDDs and flash storage

Master Files, Transaction Files, Key Fields (Serial Access)

A Master File is when you update your original magnetic tape, meanwhile a Transaction File is another tape which contains the update. The new updated tape is then referred to as the new master file, and it is vital that the records on both files are in the correct order and that is the purpose for the key field.
